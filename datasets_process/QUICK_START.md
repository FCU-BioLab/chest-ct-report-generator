# 快速導航 - Datasets Process

📖 **完整文檔請參閱**: [主 README.md](../README.md)

## 🚀 快速開始

### 資料集劃分
```bash
python split_dataset.py
```

### 推理功能
```bash
# 使用包裝器
python inference_wrapper.py --help

# 或直接使用原始推理工具
cd ../CT_ViT_Training
python inference.py --help
```

### 互動式範例
```bash
python example_usage.py
```

## 📁 工具說明
- `split_dataset.py` - 智慧資料集劃分
- `inference_wrapper.py` - 推理功能包裝器
- `example_usage.py` - 互動式使用範例
- `config_ct_vit.yaml` - 配置文件
