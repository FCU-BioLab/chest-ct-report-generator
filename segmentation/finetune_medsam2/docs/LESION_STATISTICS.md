# 有病灶切片統計功能

## 🎯 功能說明

訓練時會自動統計並保存**有病灶的切片**資訊：

1. ✅ **只保存有病灶的樣本圖片**（不再保存空白切片）
2. ✅ **統計有病灶的切片數量**
3. ✅ **計算病灶切片比例**
4. ✅ **保存統計資訊到文件**

---

## 📊 輸出內容

### 1. 樣本圖片（只有病灶的）

保存位置：`result/segmentation_*/first_epoch_samples/`

檔案命名：
- `train_positive_sample_0.png`
- `train_positive_sample_1.png`
- ...
- `val_positive_sample_0.png`
- `val_positive_sample_1.png`
- ...

**最多保存 8 個訓練樣本 + 8 個驗證樣本**

### 2. 統計資訊文件

保存位置：`result/segmentation_*/first_epoch_samples/lesion_statistics.txt`

內容範例：
```
======================================================================
有病灶切片統計
======================================================================

訓練集:
  - 總切片數: 42000
  - 有病灶切片數: 8500
  - 比例: 20.24%

驗證集:
  - 總切片數: 6000
  - 有病灶切片數: 1200
  - 比例: 20.00%

總計:
  - 總切片數: 48000
  - 有病灶切片數: 9700
  - 比例: 20.21%
```

## 💡 統計意義

### 病灶切片比例

- **正常範圍**: 15-25%（LNDb 資料集）
- **過低** (<10%): 可能資料集有問題或過濾太嚴格
- **過高** (>30%): 可能包含太多病灶切片，訓練可能過擬合

### 樣本數量

- **訓練集**: 通常 40,000-50,000 個切片
- **驗證集**: 通常 5,000-7,000 個切片
- **有病灶**: 約 20% 的切片包含病灶

---

## 🎨 樣本圖片說明

每個樣本包含 4 個子圖：

1. **原始影像**: CT 切片（中間通道）
2. **Mask (GT)**: Ground Truth 分割遮罩
3. **3-Channel**: 2.5D 視覺化（Z-1, Z, Z+1）
4. **Overlay + BBox**: 影像 + 遮罩疊加 + Bounding Box

## ⚠️ 注意事項

1. **統計會遍歷整個資料集**: 第一個 epoch 開始前會花一些時間統計
2. **只保存前 8 個樣本**: 避免生成太多圖片
3. **統計只在第一個 epoch 執行**: 後續 epoch 不會重複統計

## 🚀 使用方式

無需任何額外配置，直接訓練即可：

```bash
cd segmentation
python finetune_medsam2/main.py
```

訓練開始後會自動：
1. ✅ 統計有病灶的切片數
2. ✅ 保存有病灶的樣本圖片
3. ✅ 生成統計報告文件
